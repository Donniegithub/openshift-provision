# OpenShift Provisioner

This project is a work in progress.

## AWS

Required variables:
  - cluster_name
  - aws_region
  - route53_hosted_zone
  - route53_hosted_zone_id
  - ec2_key_name
  - ec2_key_file
  - rhsm_username
  - rhsm_password
  - rhsm_pool

Recommended variables:
  - ec2_ami_type
  - openshift_version

Required environment variables:
  - AWS_ACCESS_KEY_ID
  - AWS_SECRET_ACCESS_KEY
